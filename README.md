## Rodando localmente

Clone o projeto no seu computador
```bash
  git clone https://github.com/CassioJhones/FastAnotation.git
```

Entre no diretório do projeto
```bash
  cd pasta-do-projeto
```

Instale as dependências
```bash
  npm install
```

Inicie o servidor
```bash
  npm run dev
```
